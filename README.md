# 1. Initialization

## Install Python3.6
```
brew install python3
python3 --version
```

## Create virtual environment
```
python3 -m venv venv
```

## Activate virtual environment
```
source venv/bin/activate
```

## Install Mysql
```
brew install mysql
```


# 2. Docker run
```
make docker-up
```


# 3. Run

## Install packages and Run
```
make all
```

## Install packages
```
make install
```

## Run server
```
make run
```
